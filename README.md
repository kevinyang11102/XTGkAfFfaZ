# 高校后勤管理系统 java1002

## 项目概述

高校后勤管理系统是一款面向高校后勤管理工作的软件，旨在提高后勤服务效率和水平。系统基于SSM框架（Spring、SpringMVC、Mybatis）开发，功能丰富、逻辑清晰，适合初学者学习和掌握。

## 技术栈

### 前端技术

- JSP
- HTML
- CSS
- JavaScript
- JQuery

### 后端技术

- Spring
- SpringMVC
- Mybatis

### 开发环境

- IntelliJ IDEA

### 运行环境

- Web服务平台：Tomcat 8.0/9.0
- 数据库：MySQL 5.7/8.0
- JDK：1.8

## 功能模块

### 前台功能

- 图片咨询
- 学生咨询

### 后台管理

#### 系统设置

- 菜单管理
- 角色管理
- 密码修改

#### 用户管理

- 工人管理
- 学生管理

#### 报修与费用管理

- 报修管理
- 费用管理

#### 系统运维

- 系统日志
- 投诉管理
- 首页管理

## 系统角色

- 管理员
- 学生
- 维修工人

- 基于B/S系统的三层体系结构，实现表示层(view)、功能层(controller)、数据层(model)的分离，易于维护和扩展。
- 功能全面，覆盖高校后勤管理的主要工作内容，助力提升管理效率。
- 使用SSM框架，确保项目结构的清晰与稳定。
